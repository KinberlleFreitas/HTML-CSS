Trabalhando com Listas

Listas Ordenadas

Para criar lista ordenada uso a TAG OL para delimitar a lista e dentro dela e TAG LI para identificar item de cada lista, como exemplo abaixo;

  1. Acordar
  2. Estudar Inglês
  3. Preparar Lancheira das crianças
  4. Tomar Café
  5. Levar crianças para escola
  6. Exercico Físico
  7. Atividade Domestica
  8. Estudar Progamação

Posso personalizar a lista mudando formação da frente com numerações, e outros com parametro TYPE.

  1. Acordar
  2. Estudar Inglês
  3. Preparar Lancheira das crianças
  4. Tomar Café
  5. Levar crianças para escola
  6. Exercico Físico
  7. Atividade Domestica
  8. Estudar Progamação

Ainda dentro da tag OL posso alem de inserir parâmetro TYPE tenho o paramentro START que indica inicio da contagem, como exemplo abaixo;

  1. Acordar
  2. Estudar Inglês
  3. Preparar Lancheira das crianças
  4. Tomar Café
  5. Levar crianças para escola
  6. Exercico Físico
  7. Atividade Domestica
  8. Estudar Progamação

Listas não ordenadas

Para criar lista não ordenada uso a TAG UL para delimitar a lista e dentro dela e TAG LI para identificar item de cada lista, como exemplo abaixo;

  • Pao
  • café
  • Arroz
  • macarrao
  • lapis de cor para Yossef

Dentro da TAG UL também podemos inserir parametros para marcador padrão usando parametro TYPE + square/circle ou disc (este é padrão) , exemplo abaixo;

  • Pao
  • café
  • Arroz
  • macarrao
  • lapis de cor para Yossef

    Lista de Definição

    Usamos para inserir TERMOS e DEFINIÇÃO deste termo, usando as TAGS DT para termo e DD para definição, é como se fosse um dicionario, veja exemplo abaixo;

      HTML
      Linguagem de marcação para a criação do conteudo de um site.
      CSS
      Linguagem de marcação para criação de design de um site.
      JavaScipt
      Linguagem de programação para criação de interatividade de um site.
